## Файлы проекта
- myscript.sh - скрипт обработки аргументов командной строки
- myscript2.sh - скрипт создания файловой структуры
- cherry_pie.sh - скрипт самокопирования с логированием
- white_lodge.txt - файл для записи логов скрипта cherry.pie
- my_file.txt - список имён для директорий
- messages.txt - текстовое сообщение
